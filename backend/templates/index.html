Overview

This website is designed to provide users with motorcycle specifications, comparisons, and news on new launches. Admins can manage data via the admin portal.

Features

  • Browse bikes by brand and specs
  • Compare two or more motorcycles
  • Latest bike news and updates
  • Admin control panel for data management

Tech Stack

  • Backend: Django / Django Rest Framework
  • Database: MySQL

How to Use

Users can browse bike models and details without logging in. Admins can click the Admin Login button above to access the admin panel using their credentials.
